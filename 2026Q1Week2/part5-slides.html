The 1000px Breakpoint

Your Homework 2 Requirement

1000px
The magic number for this assignment
/* Mobile styles first (default) */
.container {
  display: grid;
  grid-template-columns: 1fr; /* Single column */
}

/* Then add desktop styles */
@media (min-width: 1000px) {
  .container {
    grid-template-columns: 1fr 1fr; /* Two columns */
  }
}
📈

The clamp() Function

Fluid Values with Guardrails

Syntax

clamp(minimum, preferred, maximum)
min ← preferred (fluid) → max

Example: Fluid Gap

gap: clamp(1rem, 3vw, 2rem);
/* At least 1rem, at most 2rem, scales with viewport */

💡 clamp() lets values "breathe" between a minimum and maximum!
