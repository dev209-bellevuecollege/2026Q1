Code Examples

Getting Started

Card Component

function Card({ title, image, description }) {
  return (
    <div className="card">
      <img src={image} alt={title} />
      <h3>{title}</h3>
      <p>{description}</p>
    </div>
  );
}

export default Card;

CardList with map()

import Card from './Card';

function CardList({ items }) {
  return (
    <div className="card-list">
      {items.map((item) => (
        <Card
          key={item.id}
          title={item.title}
          image={item.image}
          description={item.description}
        />
      ))}
    </div>
  );
}

export default CardList;

Tips for Success

Avoid Common Mistakes

Do This

  • ✓ Start with npm create vite@latest
  • ✓ Build one component at a time
  • ✓ Test in browser frequently
  • ✓ Check console for errors
  • ✓ Use React DevTools extension
  • ✓ Commit often to GitHub

Avoid This

  • ✗ Don't use array index as key
  • ✗ Don't call components as functions
  • ✗ Don't forget to export components
  • ✗ Don't modify props
  • ✗ Don't mix class and function components
  • ✗ Don't forget className (not class)
