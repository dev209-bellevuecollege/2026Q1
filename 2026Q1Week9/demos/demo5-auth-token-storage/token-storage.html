Note about this demo: True HttpOnly cookies can only be set by a server (via Set-Cookie header). Since this is a client-side demo, we simulate the concept by showing that JavaScript-set cookies ARE readable, while server-set HttpOnly cookies would NOT be. Open DevTools → Application → Cookies to see the difference.
