Arrow Functions

Cleaner Function Syntax

Traditional vs Arrow

function(x) { return x * 2; }
→
(x) => x * 2
// Full syntax
const add = (a, b) => {
  return a + b;
};

// Concise (single expression)
const add = (a, b) => a + b;

// Single param: parens optional
const double = x => x * 2;

With Array Methods

// Before (verbose)
numbers.map(function(n) {
  return n * 2;
});

// After (clean!)
numbers.map(n => n * 2);

// Chaining becomes elegant
users
  .filter(u => u.active)
  .map(u => u.name);

Arrow functions also fix this binding issues!

Template Literals

Better String Building

String Interpolation

"Hello, " + name + "!"
→
`Hello, ${name}!`
const name = "Alice";
const age = 25;

// Use backticks ` not quotes '
const msg = `${name} is ${age} years old`;

// Expressions work too!
const msg2 = `Next year: ${age + 1}`;

Multi-line Strings

// Great for HTML templates!
const card = `
  <div class="card">
    <h3>${user.name}</h3>
    <p>${user.email}</p>
  </div>
`;

Key: Use backticks ` (below Esc key), not single quotes!

Destructuring

Unpack Values Elegantly

Object Destructuring

const user = {
  name: "Alice",
  age: 25,
  email: "alice@example.com"
};

// Old way
const name = user.name;
const age = user.age;

// Destructuring
const { name, age } = user;

// In function parameters!
function greet({ name, age }) {
  return `${name} is ${age}`;
}

Array Destructuring

const colors = ["red", "green", "blue"];

// Get first two elements
const [first, second] = colors;
// first = "red", second = "green"

// Skip elements
const [, , third] = colors;
// third = "blue"

Very common in React: const [count, setCount] = useState(0)

Spread Operator

Expand and Combine

Arrays

const arr1 = [1, 2, 3];
const arr2 = [4, 5, 6];

// Combine arrays
const combined = [...arr1, ...arr2];
// [1, 2, 3, 4, 5, 6]

// Copy array (not reference!)
const copy = [...arr1];

// Add to array immutably
const withNew = [...arr1, 4];

Objects

const user = { name: "Alice", age: 25 };

// Copy and modify
const updated = {
  ...user,
  age: 26  // override age
};
// { name: "Alice", age: 26 }

// Add new property
const withEmail = {
  ...user,
  email: "alice@test.com"
};

Spread creates copies - essential for immutable state updates in React!

AI

AI as a Code Modernizer

Convert ES5 to ES6+

Before (ES5)

var users = [
  { name: "Alice", age: 25 },
  { name: "Bob", age: 30 }
];

var names = users.map(function(user) {
  return user.name;
});

var greeting = "Hello, " + names[0] + "!";

After (ES6+)

const users = [
  { name: "Alice", age: 25 },
  { name: "Bob", age: 30 }
];

const names = users.map(
  ({ name }) => name
);

const greeting = `Hello, ${names[0]}!`;
