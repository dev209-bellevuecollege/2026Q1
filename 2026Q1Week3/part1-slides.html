Objects & Arrays

Structured Data in JavaScript

Objects (Key-Value Pairs)

const person = {
  name: "Alice",
  age: 25,
  isStudent: true
};

// Access properties
person.name        // "Alice"
person["age"]      // 25

Arrays (Ordered Lists)

const colors = ["red", "green", "blue"];

// Access by index (0-based)
colors[0]          // "red"
colors.length      // 3

// Common methods
colors.push("yellow");
colors.pop();

Arrays of Objects - Very Common Pattern!

const users = [{ name: "Alice", age: 25 }, { name: "Bob", age: 30 }];

Functions

Reusable Blocks of Code

Function Declaration

function greet(name) {
  return "Hello, " + name;
}

greet("Alice");  // "Hello, Alice"

Key Differences from Java

  • No type declarations
  • All parameters optional
  • Missing params = undefined

Function Expression

const greet = function(name) {
  return "Hello, " + name;
};

// Functions are values!
// Can be stored in variables,
// passed as arguments, returned

Functions without return yield undefined

Control Flow

Making Decisions & Repeating Actions

Conditionals

if (score >= 90) {
  grade = "A";
} else if (score >= 80) {
  grade = "B";
} else {
  grade = "C";
}

Ternary Operator

// condition ? ifTrue : ifFalse

const status = age >= 18
  ? "adult"
  : "minor";

// Great for simple choices

Loops

for (let i = 0; i < 5; i++) {
  console.log(i);
}

while (condition) {
  // do stuff
}

AI

AI-Assisted Function Writing

Your New Coding Partner

Effective Prompts

"Write a JavaScript function that takes an array of numbers and returns the average. Handle empty arrays by returning 0."

Good Prompts Include:

  • Input types and format
  • Expected output
  • Edge cases to handle

AI Output

function average(numbers) {
  if (numbers.length === 0) {
    return 0;
  }
  const sum = numbers.reduce(
    (acc, n) => acc + n, 0
  );
  return sum / numbers.length;
}

Always understand what the AI wrote before using it!
